File Naming Best Practices:

  • Use lowercase letters
  • Replace spaces with hyphens (example-file-name.jpg)
  • Include descriptive names (hero-banner-ai-demo.png)
  • Add dimensions for images (profile-pic-800x600.jpg)

Image Optimization Steps:

  1. Save AI-generated images in PNG format for best quality
  2. Optimize file size using tools like TinyPNG
  3. Aim for file sizes under 500KB for fast loading
  4. Keep dimensions under 1920px width for web use

Uploading to Google Sites:

  1. Click the Images button in the right sidebar
  2. Select "Upload" from the options
  3. Choose your optimized image file
  4. Wait for upload completion
  5. Drag and drop the image onto your page

Embedding Process:

Troubleshooting üîß

Common Issues and Solutions:

Images Not Loading:

  • Verify file path is correct
  • Check image file size (under 5MB)
  • Ensure image format is supported (JPG, PNG, GIF)

HTML Not Working:

  • Remove any script tags
  • Use only inline styles
  • Check for unclosed tags

Layout Issues:

  • Test on multiple devices
  • Use relative units (%, vh, vw)
  • Add max-width constraints
